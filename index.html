Infos:

In diesem Adventskalender geht es darum, jeden Tag eine neue Aufgabe zu lösen. Die Aufgaben sind rund um Python Kara aufgebaut. Python Kara wird zur Lösung jedoch nicht benötigt. Lediglich grundlegendes Wissen über Python und das Befehlset von Python Kara sind erforderlich. (Einsteigerkenntnisse in Englisch sind zu empfehlen, aber nicht unbedingt notwendig.)

Coding im Advent ist inspiriert von Advent of Code und Mathe im Advent, hat jedoch nichts mit beiden Genannten zu tun.

Wichtiges:

  • Diese Aufgaben sind als leichte Rätsel gedacht. Bitte verwendet keine KI-Hilfsmittel, um die Aufgaben zu lösen.

  • Die Aufgaben können jederzeit ab dem jeweiligen Tag um 0 Uhr gelöst werden.

  • Alle Aufgaben der Vortage sind auch nachträglich noch einsehbar und lösbar.
