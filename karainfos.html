Benutzung Nutzen
move() Schritt vorwärts
turnLeft() 90°-Linksdrehung
turnRight() 90°-Rechtsdrehung
putLeaf() Kleeblatt hinlegen
removeLeaf() Kleeblatt aufnehmen
treeFront() Kara vor Baum?
treeLeft() Baum links von Kara?
treeRight() Baum rechts von Kara?
onLeaf() Kara auf Kleeblatt?
mushroomFront() Kara vor Pilz?
setPosition (int x, int y) Kara von Position (x,y)
getPosition() Koordinaten der aktuellen Position
Benutzung Nutzen
clearAll() Alle Elemente entfernen
setLeaf (int x, int y, boolean putLeaf) Blatt legen oder entfernen
setTree (int x, int y, boolean putTree) Baum setzen oder entfernen
setMushroom (int x, int y, boolean putMushroom) Pilz setzen oder entfernen
setSize (int newSizeX, int newSizeY) Grösse der Welt ändern
isEmpty (int x, int y) Ist das Feld leer?
isTree (int x, int y) Ist ein Baum auf dem Feld?
isMushroom (int x, int y) Ist ein Pilz auf dem Feld?
isLeaf (int x, int y) Ist ein Blatt auf dem Feld?
getSizeX() Horizontale Grösse der Welt
getSizeY() Vertikale Grösse der Welt
Benutzung Nutzen
println (String string) String auf Standard-Ausgabe schreiben
showMessage (String message) String in Dialogfenster ausgeben
random (int bound) Liefert Zufallszahl aus Bereich [0..bound]
checkState() Schaut auf den Geschwindigkeitsregler
sleep (int ms) Schläft ms Millisekunden
stringInput (String title) String in einem Dialogfenster mit Titel title eingeben; gibt null zurück, wenn der Dialog mit Cancel abgebrochen wurde
intInput (String title) Ganzzahl in einem Dialogfenster mit Titel title eingeben; gibt Integer.MIN_VALUE zurück, wenn der Dialog mit Cancel abgebrochen wurde oder nicht eine Ganzzahl eingegeben wurde
doubleInput (String title) Fliesskommazahl in einem Dialogfenster mit Titel title eingeben; gibt Double.MIN_VALUE zurück, wenn der Dialog mit Cancel abgebrochen wurde oder nicht eine Fliesskommazahl eingegeben wurde
