Wie viele Pilze und Bäume hat Kara gezählt, wenn er auf dem Blatt steht?

Schaue dir den Code an und versuche die Lösung zu finden. 

mushrooms = 0
trees = 0

while not kara.onLeaf():

    kara.turnLeft()

    if kara.mushroomFront():
        mushrooms = mushrooms + 1
    else:
        if kara.treeFront():
            trees = trees + 1

    kara.turnRight()
    kara.turnRight()

    if kara.mushroomFront():
        mushrooms = mushrooms + 1
    else:
        if kara.treeFront():
            trees = trees + 1

    kara.turnLeft()
    kara.move()

A) 0 Pilze | 4 Bäume 

B) 3 Pilze | 4 Bäume 

C) 3 Pilze | 3 Bäume 

D) 0 Pilze | 0 Bäume
